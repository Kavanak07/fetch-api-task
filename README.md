# Fetch and Display Data from Public API

## Overview
This project uses JavaScript Fetch API to retrieve user data from a public REST API and display it dynamically on a webpage.

## Objective
- Fetch data from https://jsonplaceholder.typicode.com/users
- Parse JSON response
- Display user name, email, and city
- Handle errors using try-catch
- Add reload button to refetch data

## Technologies Used
- HTML
- CSS
- JavaScript (Fetch API)

## Key Concepts
Fetch API, Promises, Async/Await, JSON parsing, REST API, Error Handling.

## Outcome
Learned how to work with APIs, handle asynchronous JavaScript, and display real-time data on a webpage.
